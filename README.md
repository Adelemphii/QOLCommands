# QOLCommands
A simple grouping of "Quality of Life" commands in one repo. Feel free to add on to this.

## Commansd
- /sit
- /roll <number>
- /broadcastroll <number>

## Permissions
- qolcommands.sit
- qolcommands.roll
- qolcommands.roll.broadcast
- qolcommands.admin
